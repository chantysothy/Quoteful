# Quoteful

#### Collection of Quotes

---

Quoteful is an iOS application where users can save a random quote and edit
them to their liking.

## Running application
    - git clone https://github.com/myleschatman/Quoteful
    - cd Quoteful
    - npm install
    - react-native run-ios
